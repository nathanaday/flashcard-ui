### 1

> A subnet mask is used to identify the boundary between network bits and host bits in an IP address

**Answer: True**

### 2

> Unlike Routers, L2 switches built their tables based on destination MAC address.

**Answer: False**

L2 switches build their MAC address table based on source MAC address, not the destination. 

### 3

> An IPv.4 packet can be fragmented and reassembled multiple times in transit.

**Answer: False**

It can be fragmented multiple times, but reassmbly only happens at the final destination.

### 4

> If Bob and Alice are two peers running behind a NAT, across a WAN, then they can not set up a TCP connection.

**Answer: False**

With nuance: practically speaking, this is possible via a number of techniques, but note that "without help" they can't communicate

### 6

> Multiple TCP sessions, on a single host, can be distinguished using IP addresses and port numbers.

**Answer: True**

TCP 4-tuple (source ip, source port, destination ip, destination port)

### 7

> SSID is a technical term for a Wi-Fi network address

**Answer: False**

SSID is the service set identifier, but not an actual address. The actual address is BSSID (typically the MAC address of the AP)

### 9

> The purpose of TCP slow start is to avoid queue overflow in routers

**Answer: True**

Practically speaking this is what it is doing and is an accepted answer.

### 10

> In 802.11 CSMA/CA, the sender can detect a collision without feedback from receiver

**Answer: False**

*wireless collision detection is impossible because radios are half-duplex, so the sender cannot detect a collision without receiver feedback (an ACK or lack thereof).*

### 11

**Answer: True**

*Matches question_bank.md #11.*

### 12

> A host sends a segment with a sequence number 50 and payload size of 50 Bytes. The ACK sequence number in that segment is 100.

**Answer: False**

The ACK number acknowledges data received from the other side and has nothing to do with data sent in the same segment.

### 13

> In an IP network, a malicious host claiming an IP address (IP address spoofing) outside its subnet will receive no packets

**Answer: True**

Generally true with nuance. When a host sends a packet with a spoofed source IP from a different subnet, the replies will be routed back to the legitimate subnet (not the malicious actor).

### 15

> Every port of a router constitutes a broadcast domain while every port of a L2 switch constitute a collision domain.

**Answer: True**

### 16

> UDP flow control guarantees the delivery of in-order datagrams to the application

**Answer: False**

UDP has no flow control whatsoever, and also no congestion control and no promise of reliable delivery.

### 17

> At the network layer, a spanning tree is used to ensure that no redundant packets are received by any node

**Answer: False**

Spanning Tree Protocol (STP) operates at Layer 2, and even then it's purpose is to prevent broadcast storms from cycles and loops.

### 18

> In 802.3 standard, if the coverage (distance) is increased, the minimum frame size would decrease.

**Answer: False**

*Under min_frame_size >= 2 * propagation_delay * bandwidth, increasing coverage increases propagation delay, so min frame size would increase, not decrease.*

### 19

> In 802.3 standard, if the bandwidth is decreased, the minimum frame size would increase

**Answer: False**

*Under the formula min_frame_size >= 2 * propagation_delay * bandwidth, decreasing bandwidth allows a smaller min frame, so the claim that it would increase is False.*

### 22

> In Wi-Fi, a wireless host needs only the MAC address of the AP. It does not need the MAC address of the default router.

**Answer: False**

A wireless host needs both MAC addresess to communicate with a destination outside of its own subnet.
